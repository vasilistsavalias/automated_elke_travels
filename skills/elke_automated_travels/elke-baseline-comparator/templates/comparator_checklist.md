# Comparator Checklist Template

Use this checklist before setting the verdict label.

## Input Parity

- [ ] Baseline and output raw file counts compared
- [ ] Extension distribution compared
- [ ] Hash multiset parity checked

## Contract Parity

- [ ] Required directory structure present
- [ ] Required artifact files present

## Output Parity

- [ ] Normalized evidence parity checked
- [ ] Submission artifact parity checked
- [ ] Divergences listed with paths and level

## Verdict

Choose exactly one:

- [ ] exact_same
- [ ] structurally_equivalent_but_content_divergent
- [ ] divergent
